# 48emote84
Add unicode-emoji support to Java Minecraft.

## How to install?
Unzip this resourse pack to new folder in /resoursepacks/

## How to use?
Press Windows + . this hotkey will activate the emoji panel.
Also, you can open onscreen keyboard.
